<article>: The Article Contents element

The <article> HTML element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication). Examples include: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submittable comment, an interactive widget or gadget, or any other independent item of content.

Attributes

This element only includes the global attributes.

Usage notes

  • Each <article> should be identified, typically by including a heading (<h1> - <h6> element) as a child of the <article> element.
  • When an <article> is nested, the inner element represents an article related to the outer element. For example, the comments of a blog post can be <article> elements nested in the <article> representing the blog post.
  • Author information of an <article> can be provided through the <address> element, but it does not apply to nested <article> elements.
  • The publication date and time of an <article> can be described using the <datetime> attribute of a <time> element.
