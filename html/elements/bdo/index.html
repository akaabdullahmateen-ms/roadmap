<bdo>: The Bidirectional Text Override element

The <bdo> HTML element overrides the current directionality of text, so that the text within is rendered in a different direction. The text's characters are drawn from the starting point in the given direction; the individual characters' orientation is not affected (so characters do not get drawn backwards).

Attributes

This element's attributes include the global attributes.

dir

The direction in which text should be rendered in this element's contents. Possible values are:

  • ltr: Indicates that the text should be written in a left-to-right direction.
  • rtl: Indicates that the text should be written in a right-to-left direction.
